Quick start analysis guide


  1. Input the list of gene IDs in Sequence IDs form

    2. Paste the list of genes in Sequence IDs form. Each line should contain one gene ID as per Phytozome v12.0 database format. You can check the allowed IDs for each plant species on the analysis page.

    For example, allowed IDs for Rice (Oryza sativa) are,

    Phytozome locus: LOC_Os01g06882
    Phytozome transcript: LOC_Os01g06882.1
    Phytozome pacID: 24120792


  2. Select plant species from dropdown menu

    3. Select plant species corresponding to the gene IDs. Currently, GenFam supports analysis of sixty plant genomes.


  3. Select plant ID type from dropdown menu

    4. Select the plant ID type for respective plant species. Please, refere step 1 for allowed IDs type for GenFam analysis. Note: Do not combine different ID types while performing enrichment analysis.


  4. Select statistical significance test from enrichment analysis

    5. Select the statistical significance test from dropdown menu. Currently, GenFam supports four tests for enrichment analysis including, Fisher exact test, Hypergeometric distribution, Binomial distribution and Chi-squared distribution, thus providing flexibility in the analysis based on the sample size and user preferences.

    We recommend using Fisher exact test, chi-square test and hypergeometric distribution for smaller datasets (< 1000 gene IDs), and binomial distribution for larger datasets.


  5. Select multiple testing correction for enrichment analysis

    6. Select the multiple testing correction test from dropdown menu. GenFam supports Bonferroni, Bonferroni-Holm and Benjamini-Hochberg tests for correcting P-values.


  6. Select Run to perform enrichment analysis

    7. Once all parameters selected, click run to perform the analysis.


  7. GenFam enrichment ouput

    8. If enrichment analysis is performed successfully, you will receive a message "Success! Data Analysis Completed".

    You can obtain results for All Enriched Families, All Gene Families and Get Figures

    "All Enriched Families" file contain results for gene families enriched with P-value < 0.05. "All Gene Families" file contain results for all gene families which are classified based on user dataset. Get figures will generate the bar graph of enriched gene families.
